NOTE: in v2.04 and later versions, an additional modification to main.F is required for the solid-state NEB.
Find and replace:

      CALL CHAIN_FORCE(T_INFO%NIONS,DYN%POSION,TOTEN,TIFOR, &
           LATT_CUR%A,LATT_CUR%B,IO%IU6)

with

      CALL CHAIN_FORCE(T_INFO%NIONS,DYN%POSION,TOTEN,TIFOR, &
           TSIF,LATT_CUR%A,LATT_CUR%B,IO%IU6)

SVN Access
==========

A read-only copy of the code can be anonymously checked out of the subversion
repository with the following command:

.. code-block:: sh

    svn co http://theory.cm.utexas.edu/svn/vtsttools

If you are a developer you can checkout the code using the command:

.. code-block:: sh

    svn co svn+ssh://username@theory.cm.utexas.edu/svn/vtsttools

This will fetch a copy of the latest code to a local directory named eon.
For more information on subversion read its `online documentation`_.
